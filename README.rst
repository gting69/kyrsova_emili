Coworking Management
====================
A module for automating the work of a co-working center.
Description
----
This application allows you to manage halls, workplaces and additional services.
Includes a reservation system with different types presented (calendar, kanban, pivot tables).
Functional
----------
* Accounting for rooms and tables.
* Flexible tariff setting.
* Wizard for quick creation of records.
* Print PDF booking confirmation.
